
The images in the wiki are stored locally here under a directory name that mirrors the wiki entries path. However, the embed images are pointing to the [[git/GitHub|GitHub]] repo using the images url and the query string `?raw=true` eg, `https://github.com/nickmedwards/ESOL_wiki/blob/main/images/README/default_branch_name.png?raw=true`. 

The images are then embed by exclamation point, square brackets around the alt text, parentheses around the image url ie, `![alt text](image_url?raw=true)`.
