This project is developing a CSP plant operator training simulation modeled in Fortran using TRNSYS and supporting real-time web-enabled interaction.
This project was a collaboration between [Anna](https://github.com/annagroeschael), [Chris](https://github.com/volkvein), [Matt](https://github.com/tuman), [Nick](https://github.com/nickmedwards), and [Rob](https://github.com/mcfierce), with support from [Sergio](https://gihub.com), [John](https://gihub.com) from Isentropic and [Keith](https://gihub.com) from Solar Dynamics.
